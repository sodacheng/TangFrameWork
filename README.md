完成所有项目都有的公共模块
- 为什么先做这些?
   - 公共模块可以简单的理解为整个程序框架,它提升开发效率
   - 这些模块在游戏中各处都会用到
   - 往往这些模块在各个小项目中是通用的,完成一次后可通用
- 包含内容
   - 单例模式基类
   - 缓存池模块
   - 事件中心模块
   - 公共Mono模块
   - 场景切换模块
   - 资源加载模块
   - 输入控制模块
   - 音效管理模块
   - UI模块
   - 数据管理模块
